# Simulador de Promociones Mercado Libre

App en Streamlit para cruzar promociones de Mercado Libre con una base de Tienda Nube.

## Funciones principales

- Subir Excel de promociones de Mercado Libre.
- Subir CSV de Tienda Nube.
- Cruzar datos por SKU.
- Mostrar costo, EAN, descripción, categoría y marca.
- Editar descuento y precio final.
- Marcar automáticamente ACTION como Participar cuando se modifica descuento o precio final.
- Ordenar productos por categoría/familia y SKU.
- Exportar archivo final respetando la plantilla original de Mercado Libre.
